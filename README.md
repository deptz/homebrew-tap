# homebrew-tap

Homebrew tap for Deptz repositories.

## Available tools

| Tool | What it is |
|------|------------|
| **augflow** | Local-first developer productivity tool. |

## Install

```bash
brew tap deptz/homebrew-tap
brew install augflow
```

---

Formulas in `Formula/` are updated automatically by [GoReleaser](https://goreleaser.com) when a release is published. Prebuilt binaries are hosted on [deptz/augflow-releases](https://github.com/deptz/augflow-releases).
